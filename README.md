## 🚀 Projects

### [CanvasLink — 타투이스트 예약 관리 SaaS](https://canvaslink.kr/)
타투이스트를 위한 예약·상담·동의서·시술 기록 통합 관리 웹 서비스. 2027년 시행되는 문신사법 대응을 핵심으로 설계.

- 채팅 기반 1:1 상담 → 예약 신청 → 미성년자 자동 검증 → 전자 동의서 → 입금 확인까지 이어지는 예약 풀 플로우 구현
- 문신사법 제18조(시술 기록)·제19조(동의서)·제22조(미성년자 보호) 대응 기능을 제품 핵심 가치로 설계
- Firestore 실시간 구독 기반 대시보드/캘린더/채팅 동기화, Paddle 글로벌 구독 결제 연동
- Instagram: [@canvaslink_official](https://instagram.com/canvaslink_official)

**Stack**: React (Vite), Firebase, Cloud Functions, Paddle

### [Hameday — 커플 프리미엄 다이어리](https://hameday.kr/)
커플의 일상과 순간을 매거진 형태로 기록하고 실시간 공유하는 프라이빗 아카이빙 웹 서비스.

- 기념일 자동 계산, 버킷리스트 관리, 감정 공유 기능 제공
- Vanilla CSS 기반 Glassmorphism UI + Framer Motion 애니메이션으로 프리미엄 UX 구현
- i18next 다국어 지원 및 Paddle 글로벌 결제 연동으로 해외 확장성 확보
- Node.js 기반 커스텀 하네스 스크립트로 코드 표준 및 보안 검증 자동화

**Stack**: React (Vite), Firebase, Vanilla CSS, Framer Motion, i18next
